🚀 快速開始 (Quick Start)

node-pg-trigger 讓您的 PostgreSQL 資料庫具備即時推送能力。本指南將引導您在 5 分鐘內完成配置並開始監聽資料變動。

1. 安裝套件

npm install node-pg-trigger

2. 初始化資料庫設施

在後端啟動時，調用 runMigrations 自動建立 Audit Log 表與 Trigger 函式：

import { runMigrations } from 'node-pg-trigger/server';

await runMigrations();

3. 為資料表啟用監聽

執行以下 SQL 為您的目標資料表綁定觸發器：

CREATE TRIGGER posts_notify_trigger
AFTER INSERT OR UPDATE OR DELETE ON posts
FOR EACH ROW EXECUTE FUNCTION notify_db_event();

4. 前端開始訂閱

使用 Firestore 風格的語法即時獲取數據：

import { createSdk } from 'node-pg-trigger';

const sdk = createSdk('http://localhost:3000');
sdk.collection('posts').onSnapshot(snapshot => {
  console.log('即時數據:', snapshot.record);
});
